Sahi - απλό, παραγωγικός

Το Sahi είναι ένα εργαλείο αυτοματοποίησης και δοκιμής για τις εφαρμογές Ιστού, με τη δυνατότητα που καταγράφουν και τα χειρόγραφα αναπαραγωγής ήχου. Το Sahi τρέχει σε οποιαδήποτε σύγχρονη μηχανή αναζήτησης που υποστηρίξεις javascript.

Αναπτυγμένο στην Ιάβα και javascript, Sahi χρησιμοποιεί το απλό javascript για να εκτελέσει τα γεγονότα στη μηχανή αναζήτησης.

Χαρακτηριστικά γνωρίσματα:

  • έλεγχοι -μηχανών αναζήτησης
  • ευφυείς αρχείο και αναπαραγωγή ήχου
  • Δυνατότητα να προστεθούν οι ισχυρισμοί (σημεία ελέγχου) για την επικύρωση
  • το κείμενο βάσισε τα προγραμματίσημα χειρόγραφα με τη δυνατότητα οι μεταβλητές, επαν-παράγοντας στις λειτουργίες και οργανώνει και περιλαμβάνει άλλα χειρόγραφα
  • δυνατότητα να τρεχτούν τα χειρόγραφα κατά δεσμίδες για την αυτοματοποιημένη δοκιμή
  • καλή ολοκλήρωση με τα εργαλεία κατασκευής μέσω των στόχων μυρμηγκιών και της γραμμής εντολής
  • αυτόματο HTML που υποβάλλει έκθεση με τα κούτσουρα λάθους
  • εύκολα εκτατός μέσω των απλών λειτουργιών javascript για να εξυπηρετήσει στους ιδιαίτερα δυναμικούς ιστοχώρους
  • βάση δεδομένων, αρχείο ανάγνωσες-γραφής APIs για προσανατολισμένη προς την τα στοιχεία δοκιμή
  • η ταυτόχρονη εκτέλεση των δοκιμών για μειωμένος χτίζει το χρόνο (πολύπλοκη αναπαραγωγή ήχου)
  • Υποστήριξη HTTP και HTTPS
  • άριστη υποστήριξη AJAX

Το Sahi τρέχει ως κεντρικός υπολογιστής πληρεξούσιου και η μηχανή αναζήτησης πρέπει να χρησιμοποιήσει τον κεντρικό υπολογιστή Sahi ως πληρεξούσιό της. Το Sahi εγχέει έπειτα javascript έτσι ώστε μπορεί να έχει πρόσβαση στα στοιχεία στο webpage. Αυτό κάνει τον ανεξάρτητο εργαλείων της μηχανής αναζήτησης εφαρμογής Ιστού ιστοχώρου.

Το Sahi απελευθερώνεται με άδεια Apache (έκδοση 2.0)
